## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Pratham54301/Hoobank-.git
cd Hoobank-
````

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```
